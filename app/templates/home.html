Player Comparison

This section allows you to compare stats between 2 players for the 2019-2020 season

Using an API from Rapid API (https://rapidapi.com/theapiguy/api/free-nba) here to query all available players based on a user search, selecting them to get all of stats found using balldontlie (https://www.balldontlie.io/) API. After getting the stats using Javascript fetch and return the stats in json format. From there, I utilized the open source echarts library to visualize the stats side by side.
